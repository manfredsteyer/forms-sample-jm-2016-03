# Angular-2-Beispiel: Nutzung von Formularen

Dieses Beispiel zeigt die folgenden Aspekte von Angular 2:

- Nutzung deklarativer Formulare ("template-driven Forms")
- Nutzung imperativer Formulare
- Benutzerdefinierte Validatoren für deklarative Formulare
- Benutzerdefinierte Validatoren für imperative Formulare
- Asynchrone Validatoren
